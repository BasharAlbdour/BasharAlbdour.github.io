## What's inside
 
Five project phases built over the course of the AI.SPIRE program:
 
- **Foundations** (Modules 1–3) — Git workflows, data pipelines, PyTorch fundamentals, SQL analytics, and ETL pipeline design
- **ML & NLP** (Modules 4–7) — Data visualization, machine learning models, NLP pipelines, and deep learning with transformers
- **Retrieval & Knowledge Graphs** (Modules 8–9) — RAG systems, vector search, and knowledge graph design
- **Deployment** (Modules 10–11) — Production APIs, containerisation, and monitoring
- **Capstone** (Module 12) — End-to-end AI/ML system
